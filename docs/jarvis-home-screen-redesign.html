שקף 07 — טיפול במקרי קצה

Edge Cases & Resilience

🧊

Cold Start — משתמש חדש

Onboarding מוטמע ב-Hero Card (לא מסך נפרד): 2 שאלות בלבד. רק Weather + Tasks גלויים. לאחר 3 ימים — AI mode מלא.

📵

Offline Mode

Tasks + Reminders מ-local cache. Weather/News/LLM — greyed-out עם "זמין עם חיבור". Banner amber דיסקרטי (לא alert).

🌊

Information Overload

Top-3 per category + badge counter. ≥3 תזכורות דחופות → "Urgent Mode": Hero Card הופך לרשימה בלבד. DND Mode: snooze 2 שעות.

🔒

Privacy Mode

Long-press על Avatar. מסתיר: שמות, סכומים, תוכן הודעות. Auto-Privacy: חוסר מגע >3 דק בשעות עבודה. נשמר ב-/user-profile.

async function extractUserPatterns(supabase) {
  const { data } = await supabase
    .from('chat_history')
    .select('text, intent, created_at')
    .order('created_at', { ascending: false })
    .limit(200);

  // cluster by hour + day + intent
  return clusterByTimeAndIntent(data);
  // { tuesday_morning: ['stocks'],

}
