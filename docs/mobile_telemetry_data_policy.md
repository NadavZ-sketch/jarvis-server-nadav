# Mobile Telemetry Data Policy

## מה נאסף
- מזהה משתמש פסאודונימי (hash מקוצר), לא שם חופשי.
- שם אירוע מתוך רשימת enums ידועה מראש.
- מונה/ערך מספרי בלבד.
- timestamp לכל אירוע.
- metadata מוגבל לשדות קטגוריאליים (למשל `compactMode`, `action`).

## למה נאסף
- למדוד שימוש במסך ההתקדמות.
- לשפר UX ולקבל החלטות Roadmap.

## לכמה זמן נשמר
- אירועי שימוש רגילים: 30 יום.
- אירועים רגישים (לדוגמה outcome של הצעה): 90 יום.

## איך מוחקים
- מחיקה אוטומטית בצד שרת לפי TTL.
- מחיקה יזומה: כפתור "איפוס למידה / Telemetry" במסך ההגדרות.
